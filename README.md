# ROY MART

ROY MART — a modern, modular e-commerce marketplace scaffold (React + Vite frontend, Node + Express backend, MongoDB). This repository is scaffolded for development in GitHub Codespaces.

Quick start (Codespaces / local):

1. Copy .env.example to .env in the repository root and edit values as needed.
2. From the repository root, install dependencies and start services:
   - Install root helper deps: npm install
   - Install client & server: cd client && npm install; cd ../server && npm install
3. Run client and server in separate terminals (or use root scripts):
   - npm run dev:client
   - npm run dev:server

Client:
- Vite + React + Tailwind CSS
- Pages: Home (built), others scaffolded

Server:
- Node + Express
- MongoDB connection helper
- Basic API routes scaffolded
